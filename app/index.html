Health Neighbourhoods Postal Code Lookup!

Map actions explained

There are several things you can do with this map:

  • Hover the blue lines to identify a Health Neighbourhood (see the information box on the bottom right)
  • Hover within a Health Neighbourhood to identify the postal code(s) within the boundaires
  • Click a Health Neighbourhood to toggle the border colour (useful for isolating it from the others!)
  • Add/Remove a layer from the map using the dialog in the top right
